    Project Description

    Bicycle Trek was a project started at the CodeItForward Hackathon in July 2015. The goal of the project was to develop a mobile application for BC Lung Association's annual Bicycle Trek fundraiser. Specifically, it is an exercise and training application designed for the participants of the event to help them track their progress while training. In addition, the application is designed to incoporate social media outlets to raise awareness for this event and attract a younger demographic to participate.

    Responsibilities

  • Implemented a millisecond stopwatch timer to be used to record training duration
  • Implemented a countdown timer to tell users how many days until the event is left
  • Used Facebook, Twitter and Google APIs to allow social media access to the application

    Project Description

    Parkcouver was a web application project designed completely from the ground up using Django Python. The goal of the project was to use a given set of data and create a web application that allows the client to interact with the server to read and write data. The dataset of interest was parks in Vancouver and as users of the application, one can search and locate parks closest to their location, discover what facilities parks offer and keep track of parks they like.

    Responsibilities

  • Used Facebook and Twitter APIs to allow for social media login, Facebook shares, likes and tweeting
  • Implemented methods to let users "favourite" a park and have the data updated in the database
  • Designed the frontend of the application using JavaScript, HTML and CSS
